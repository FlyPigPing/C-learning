# 文档
## json

**1.什么是JSON？**

JSON是存储和交换文本信息的语法。类似于XML。

JSON比XML更快、更小，更容易解析。JSON跟XML一样是数据格式。

JSON是轻量级的文本数据交换格式，独立于语言，具有自我描述性，更容易理解

**2.JSON语法规则**

数据在建值对中

```
"firstName":"John"
"result":123
"flag":true
"object":{key:value,key:value}//对象
"int"：["java","javascript"]//数组
```

数据由逗号分割

花括号保存对象

方括号保存数组
